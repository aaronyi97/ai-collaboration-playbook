# 中文视频置顶评论版

这是给视频置顶评论、简介区或社群消息使用的短版。观众复制下面整段到 OpenClaw、Cursor、Claude Code、Codex 或其他 AI 编程工具里即可。

不要只让观众复制 GitHub 链接。很多 AI 工具不会自动读取链接，容易直接吐出一大段错误结果。

```text
我想给当前项目生成一张 AI 接棒卡，用来把旧对话/旧任务交给新 AI 继续。

请按这个开源 Kit 的方法做：
https://github.com/aaronyi97/ai-collaboration-playbook/tree/main/kits/001-context-handoff

重要规则：
1. 你不要现在写接棒卡。
2. 第一轮只问我 6 个问题，最多 12 行。
3. 不要输出极简版、完整版、解释版，也不要一次输出三份文档。
4. 如果你打不开 GitHub 链接，也继续按下面的短版流程做；不要停止，不要要求我先粘贴其他文件。

第一轮只问这 6 个问题：
1. 这个项目/任务的目标是什么？
2. 现在做到哪一步？
3. 已经完成了什么？
4. 还有什么没完成？
5. 哪些方案、文件或方向不要再碰？
6. 下一个 AI 第一件事应该做什么？

我回答后，你先给我一版不超过 12 行的用户预览版，让我能看懂接棒卡准备写什么。

我确认后，你只输出一段最终可复制版，放在一个代码块里，并写清楚“请把下面整段复制到新 AI 对话”。默认控制在 40 行以内。

只有当我明确说“给完整版”时，你才需要让我补充完整模板或打开 GitHub Kit。
```
