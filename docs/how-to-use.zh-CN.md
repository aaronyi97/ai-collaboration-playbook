# 使用说明

## 给 AI 编程工具使用

每个 Kit 里都有一个 `START.zh-CN.md`。

这份启动 Prompt 默认你正在使用 OpenClaw、Cursor、Claude Code、Codex 这类 AI 编程工具。它会要求 AI：

1. 先读 Kit 文档；
2. 询问缺失的项目信息；
3. 不要过早改代码；
4. 生成本期对应的可复用产物；
5. 给出可以复制到新对话的接手 Prompt。

## 给普通用户使用

如果你不熟 GitHub，不用看完整仓库。打开视频置顶评论里的 `START.zh-CN.md`，复制里面那段话，粘贴到你的 AI 编程工具里即可。

## 推荐流程

```text
看视频
-> 打开对应 Kit
-> 复制 START.zh-CN.md
-> 粘贴到 OpenClaw / Cursor / Claude Code / Codex
-> 回答 AI 问你的问题
-> 生成自己的项目交接卡
-> 下次换对话时直接用
```

## 边界

这些 Kit 是实用模板，不是生产级运行时系统。它们帮你把 AI 协作流程整理清楚，但不能替代测试、代码审查、安全检查和人的判断。
