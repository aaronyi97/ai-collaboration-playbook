# 使用说明

## 给 AI 编程工具使用

每个 Kit 里都有一个 `START.zh-CN.md`。

这份启动 Prompt 默认你正在使用 OpenClaw、Cursor、Claude Code、Codex 这类 AI 编程工具。它会要求 AI：

1. 先读 Kit 文档；
2. 根据当前对话和项目状态直接生成简版接棒表；
3. 不要过早改代码；
4. 把可复用内容放进代码块，方便复制；
5. 只有用户需要时，再生成详细版接棒卡。

## 给普通用户使用

如果你不熟 GitHub，不用看完整仓库。打开视频置顶评论里的 `PINNED.zh-CN.md`，复制代码块里的整段话，粘贴到你的 AI 编程工具里即可。

不要只把 GitHub 链接丢给 AI。很多 AI 工具不会自动读取链接，它可能会直接生成一大篇错误文档。

## 推荐流程

```text
看视频
-> 打开对应 Kit
-> 复制 PINNED.zh-CN.md 代码块
-> 粘贴到 OpenClaw / Cursor / Claude Code / Codex
-> AI 直接生成简版接棒表
-> 需要时再生成详细版
-> 下次换对话时直接用
```

## 边界

这些 Kit 是实用模板，不是生产级运行时系统。它们帮你把 AI 协作流程整理清楚，但不能替代测试、代码审查、安全检查和人的判断。
